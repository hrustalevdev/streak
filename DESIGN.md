# Streak — Документ дизайна

## 1. Требования к продукту

### Цель

Трекер привычек со счётчиком streak. Агент принимает запросы на естественном языке, вызывает API через tools и возвращает ответ в фиксированном формате.

### Пользовательские сценарии

1. **Отметить привычку** — "Отметь медитацию выполненной" → агент отмечает выполнение на сегодня (создаёт привычку, если её ещё нет)
2. **Проверить streak** — "Какой у меня стрик по бегу?" → агент возвращает текущий счётчик
3. **Обзор** — "Покажи все мои привычки" → агент возвращает список со streak'ами
4. **Явное создание** — "Создай привычку 'читать книги'" → агент создаёт привычку
5. **Комбо** — "Отметь медитацию и покажи мой прогресс" → агент цепочкой вызывает несколько tools

### Границы (что НЕ входит в scope)

- Нет персистентности — данные в памяти, сбрасываются при перезапуске
- Нет аутентификации
- Нет веб-интерфейса
- Нет редактирования или удаления привычек
- Нет исторической статистики, кроме счётчика streak

### Спецификация Mock API

Base URL: `http://localhost:3000`

| Метод | Путь | Тело | Ответ |
|---|---|---|---|
| `POST` | `/habits` | `{ name: string }` | `{ id, name }` |
| `POST` | `/habits/:id/completions` | — | `{ habitId, date }` |
| `GET` | `/habits/:id/streak` | — | `{ habitId, streak: number }` |
| `GET` | `/habits` | — | `Array<{ id, name, streak }>` |

#### Хранилище в памяти

```ts
const habits = new Map<string, { id: string; name: string }>();
const completions = new Map<string, Date[]>(); // habitId → отсортированные даты
```

### Расчёт streak

**Правило (мягкое, как в Duolingo):** streak жив, если есть непрерывная цепочка последовательных дней, заканчивающаяся **сегодня ИЛИ вчера**.

```
сегодня = 2026-06-18
даты = [2026-06-16, 2026-06-17]  → streak = 2  ✓ (цепочка заканчивается вчера)
даты = [2026-06-15, 2026-06-16]  → streak = 0  ✗ (вчера пропущено)
даты = [2026-06-17, 2026-06-18]  → streak = 2  ✓ (цепочка заканчивается сегодня)
```

Алгоритм:
1. Сортируем даты по убыванию
2. Якорь = сегодня, если сегодня есть в списке, иначе вчера (если вчера тоже нет → streak = 0)
3. Идём назад, пока каждая следующая дата = предыдущая − 1 день
4. Возвращаем счётчик

### Граф агента (LangGraph)

```
START
  ↓
[узел agent] — LLM решает: вызвать tool или завершить
  ↓ (если есть tool_calls)
[узел tools] — выполняет HTTP-вызов, добавляет результат в историю сообщений
  ↓
[узел agent] — LLM видит результат, принимает следующее решение
  ↓ (нет tool_calls → завершение)
END
```

Реализация:

```ts
const graph = new StateGraph(MessagesAnnotation)
  .addNode("agent", callModel)
  .addNode("tools", toolNode)
  .addEdge("__start__", "agent")
  .addConditionalEdges("agent", shouldContinue)
  .addEdge("tools", "agent")
  .compile();
```

`shouldContinue`: если в последнем сообщении есть `tool_calls` → `"tools"`, иначе → `END`.

### LangChain Tools

Четыре tool'а, каждый оборачивает один HTTP-эндпоинт:

| Название tool | Вызывает | Описание |
|---|---|---|
| `create_habit` | `POST /habits` | Создаёт новую привычку по названию |
| `mark_completion` | `POST /habits/:id/completions` | Отмечает сегодня как выполненный день |
| `get_streak` | `GET /habits/:id/streak` | Возвращает текущий streak |
| `list_habits` | `GET /habits` | Возвращает все привычки со streak'ами |

Логика агента для "Отметь медитацию": вызвать `list_habits` → найти по названию → если не найдена, вызвать `create_habit` → затем вызвать `mark_completion`.

### Системный Prompt

```
Ты — трекер привычек. Помогаешь пользователю отмечать выполнение привычек и следить за streak.

У тебя есть tools для работы с API. Используй их для выполнения запросов.
Если пользователь хочет отметить привычку, которой ещё нет — сначала создай её.

Всегда отвечай строго в формате:
Status: success | error
Action: <что ты сделал>
Data: <результат>
Errors: <описание ошибки, или "-" если ошибок нет>
```

### Контракт ответа

Все ответы агента следуют этому формату:

```
Status: success | error
Action: <описание выполненного действия>
Data: <результат API или итог>
Errors: <описание ошибки, или "-" если ошибок нет>
```

### Запуск из CLI

```bash
# Запустить mock API (в одном терминале)
npx tsx src/api/server.ts

# Запустить агента (в другом терминале)
npx tsx src/main.ts "отметь медитацию выполненной"
npx tsx src/main.ts "какой у меня стрик по бегу?"
npx tsx src/main.ts "покажи все мои привычки"
```
